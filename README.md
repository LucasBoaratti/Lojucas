# Lojucas
Seja bem-vindo(a) à Lojucas, um sistema de aluguel de jogos desenvolvido em C. Aqui, você pode alugar até 3 de seus jogos favoritos por um determinado número de dias!

# Instalação e uso:

1. Abra o arquivo main.c;

2. Copie o código da Lojucas;

3. Abra o site [GDB online](https://www.onlinegdb.com/#);

4. Agora, cole o código na IDE;

5. Escolha a linguagem C no canto superior direito;

6. Clique em Run para rodar o código.

Pronto! Agora você pode ver como a Lojucas funciona! Boas compras!
